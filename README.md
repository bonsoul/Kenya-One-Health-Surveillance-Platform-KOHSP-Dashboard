# Kenya-One-Health-Surveillance-Platform-KOHSP-Dashboard
*Public health surveillance application with role-based access, interactive maps, charts, and case management features.*

### Dashboard Overview
  #### KOHSP implementation includes:
   1. Role-Based Access Control: Three tiers (National Command Center, County COHU, Sub-County/Facility)
   2. Interactive Mapping: Leaflet.js integration for geographic disease tracking
   3. Data Visualization: Chart.js for epidemic curves and sector distribution
   4. Real-time Alerts: Priority-based notification system (Critical/High/Medium)
   5. Case Management: Investigation workflows with exposure history tracking
